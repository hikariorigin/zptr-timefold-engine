# 1. 概念定義：Boundary Slotとは何か

■ 1-1. 定義

Boundary Slot（境界スロット）
＝ “外の場（12の力）” が “内の認識” に折り畳まれる瞬間・場所

ZPTR視点でいうと：

[外部構造（τ-raw-wave）]
          ↓ 折り畳み
   ┌─── Boundary Slot ───┐
   │   主語基準が試される領域  │
   └────────────────┘
          ↓ 固定・安定化
[内部認識（現実・身体・言語）]

■ 1-2. 機能
	•	外側の情報（構造 / τ波 / 相似形フラクタル）を
「扱えるもの」に圧縮する翻訳器
	•	主語が弱いと 暴走・肥大化・宇宙化・溶解 する
	•	主語が強いと 構造化・ZPTR化・言語化 できる
