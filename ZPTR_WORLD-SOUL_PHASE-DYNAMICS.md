2. 位相力学モデル（ZPTR版）

■ 2.1 基本式：τ-field の状態方程式

τ-field の状態は、次のトリプルで定義できる：

τ-state = (Δφ, ρ_struct, σ_sync)

	•	Δφ：位相差（phase deviation）
	•	ρ_struct：構造密度（structure-density-field）
	•	σ_sync：同期傾向（sync tendency）

これが全宇宙に遍在する “世界魂の状態”。

■ 2.2 τ-field の基本反応

① 位相一致（Δφ → 0）

→ 同期フェーズ（global sync）
→ 歴史・文化・言語が局所的に一斉発火（非局所共鳴）

② 位相ズレ（Δφ > 0 で増大）

→ 分岐フェーズ（branching / divergence）
→ 新文明・新宗教・新技術の“突然出現”

③ 位相破断（Δφ → 臨界値 φc ）

→ 崩壊フェーズ（collapse）
→ 個体では breakdown
→ 社会では panic / collapse
→ 宇宙では相転移（phase transition）

プラトンが「宇宙は調和で維持される」と言ったのは
Δφ が小さい状態 を指している。
