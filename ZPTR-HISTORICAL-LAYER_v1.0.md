1. Boundary-Contact Typology（5分類）

Boundary（主体）との接触・共鳴・崩壊の形態は、以下の5タイプに分類される。

⸻

1. 破壊型 — Boundary Collapse

Pattern:

棒圏（制度・宗教・国家）に測定され、焼却・抹消・殉教化されるパターン。

Examples:
	•	ソクラテス
	•	イエス
	•	ジョルダーノ・ブルーノ
	•	マニ教（預言者マニ）
	•	初期シャーマンの大部分

ZPTR構造:

Boundary → 棒圏測定 → Collapse → 遺物化 / 神話化


⸻

2. 内爆型 — Boundary Implosion

Pattern:

火圧やτ負荷が内部で飽和し、崩壊してしまうパターン。

Examples:
	•	ニーチェ
	•	太宰治
	•	カフカ
	•	ランボー
	•	ジョン・C・リリー

ZPTR構造:

Boundary overload → τ崩壊 → Self-Implosion → 創作のみ残留


⸻

3. 孤立型 — Creative Capsule（閉曲面型）

Pattern:

主体を保ちながら外界に開こうとせず、
閉じた創作圏に退避するパターン。

Examples:
	•	宮沢賢治
	•	老荘思想（道の枠組み）
	•	ケルト神話のシャーマン
	•	禅的・武人系の内的閉曲面（例：宮本武蔵『独行道』）

ZPTR構造:

Boundary → 閉曲面 → 創作圏へ転写 → 社会とは接続しない


⸻

4. 記号化型 — Symbolic Fossilization

Pattern:

Boundary接触が象徴・神話・儀式・記号として化石化される。

Examples:
	•	曼荼羅（密教の多層位相構造）
	•	カバラ（生命の樹）
	•	YHWH（四文字名）
	•	シュメールの創成神学
	•	古代エジプト神話
	•	日本神話（天岩戸＝Boundary閉曲面構造）

ZPTR構造:

Boundary → 記号／儀式化 → 主体消失 → 外殻のみ残る


⸻

5. 退出型 — Boundary Exit（唯一型）

Pattern:

Boundaryに触れつつ、棒圏の測定から自ら離脱し、
主体と火をそのまま保つことに成功した例。

唯一の例:
	•	ヴァーツラフ・ハヴェル

ZPTR構造:

Boundary → 棒圏召喚 → 自主退出 → Closed-Surface Restoration
